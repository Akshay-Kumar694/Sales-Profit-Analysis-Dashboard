# Sales-Profit-Analysis-Dashboard

This file shows a Tableau dashboard that visualizes key business KPIs using multiple charts. It includes:

Sales by Segment/Region (bar chart)

State-wise Profit (filled map)

Profit by Sales (scatter plot)

Sales by Region (stacked bar chart)

Sales by Category (pie chart)

The dashboard combines geographic, categorical, and numerical insights to help understand overall sales distribution, profitability, and performance trends across categories, regions, and customer segments.
